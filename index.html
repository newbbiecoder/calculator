Instructions


  • Basic Operations : Use the number keys (0-9) and arithmetic operators (+, -, ×, ÷) to perform calculations.

  • Equal Button (=) : Click the = button to evaluate the expression.

  • Clear Entries:
    • Press r key for CE (Clear Entry) - clears the last entered number.
    • Press c key for C (Clear All) - clears the entire calculation.

  • Special Functions:
    • Press p key to insert the π (pi) value (3.14…).
    • Press u for the √ (square root) function.

  • Backspace Function: Press the Backspace key to delete the last entered digit.
